Controller Guide

◈ Getting Connected

Open the Controller Session window in the desktop app. You’ll see a Tunnel URL, Code, and PIN. Enter all three on the login screen and click Connect. Only one controller can be connected at a time — logging in kicks any previous session.

◎ Playback Controls

Skip — jumps to the next video. Clear Queue — removes all waiting videos (current keeps playing). Panic — immediately stops everything. Extend — adds minutes to the session timer. Max Time — sets a hard session limit.

◎ Sliders

Volume (0–200) and Speed (0.25–1.5×) adjust in real-time during playback. Changes show as a brief notification on the video screen.

🔒 Lock / Unlock

Lock disables the panic hotkey on the desktop — the user cannot stop playback until you unlock it. The 3-second emergency hold (Ctrl+Shift+Esc held for 3s) always works as a safety override regardless of lock state.

⚙ Session Options

Loop — restart playlist when finished. Shuffle — randomize play order. No Repeat — skip already-watched URLs. Show Cursor — keep mouse visible during playback. Min Session Lock — disable panic for N seconds at session start. Max Videos — stop after N videos.

✦ Marquee

Send text that overlays on the playing video. Use the quick phrases or type custom text. Top/Bottom controls the position.

⏱ Schedule

Start Now — immediately begins playlist playback. Schedule at Time — set a one-time start at HH:MM. Idle Auto-Start — begins when the desktop has been idle for N minutes. Weekly Schedule — set recurring day+time entries. Each entry can optionally have its own playlist — click the 📋 icon to assign URLs. Leave empty to use the default playlist.

≡ Playlist & Queue

Queue — videos waiting to play next (top section). Playlist — the master list of URLs. Use Bulk Add to paste multiple URLs at once. Use checkboxes for bulk delete. Drag entries to reorder. Click Play Now to start the playlist immediately.

↻ Reset

The Reset All button restores every setting to defaults: volume 100, speed 1.0×, no marquee, panic unlocked, loop/shuffle/repeat off.

🛡 Connection & Security

All traffic is encrypted via Cloudflare HTTPS tunnel. The tunnel URL changes on every app restart. Session codes and PINs are generated fresh and never stored on disk. If you close the browser or lose connection, the session auto-expires within 15 seconds. Only one controller can be connected at a time.
